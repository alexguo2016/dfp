我对Vue的认识

2018-6-30
Vue似乎是完全面向对象的, 至少, 可以使用面向对象的方式来理解, 我改写以前的JavaScript为Vue版本, 只需要按照对象的写法就好
Vue实例里面都包含data, methods, 生命周期方法
当data改变的时候, 页面上面相应的元素内容发生改变, 使用虚拟DOM技术进行的映射, 所以说, 每次改变, 都是生命周期里面change函数引起的
